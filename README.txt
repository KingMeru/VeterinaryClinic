PROGRAM DESCRIPTION: This program manages the animal and customer data for a veterinary clinic.
                     The end user will be able to print all the clinic's data or the animal data
                     or customer information only.

Source file(s): main.cc, Control.cc, View.cc, Clinic.cc, Animal.cc Customer.cc, Collection.cc,
                Identifiable.cc, IdArray.cc, IdList.cc, Control.h, View.h, defs.h, Clinic.h, Animal.h,
                Customer.h, Collection.h, Identifiable.h, IdArray.h, IdList.h, Makefile

Compilation Instruction: make

Launching Instruction: ./a04

Operating Instructions:
1. follow compilation instructions
2. follow launching instructions.
3. select 1 to print clinic's data.
   select 2 to print animals only.
   select 3 to print customers only.
4. Select 0 to exit the program
